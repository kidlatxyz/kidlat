papano mag simula sa bitcoin lightning wallet

Ang pinakamadaling paraan is gumawa ng custodial wallet katulad ng Pouch: The PHP Lightning Wallet o Wallet of Satoshi

Ano ang satoshi

Ang satoshi ang smallest unit ng bitcoin. Karaniwan ginagamit ang term na satoshi sa Lightning network ni Bitcoin dahil usually maliliit lang na halaga ang nilalagay at ginagamit dito.

Pano magkaroon ng satoshis gamit ang BlueWallet

Kailangan mo ng wallet katulad ng BlueWallet para loadan muna ng Bitcoin. Gamit ang BlueWallet, gagawa ka ng dalawang wallet - yung isa ay para sa regular bitcoin ang isa ay para sa lightning. Pagkatapos, i-loload mo ang Bitcoin wallet gamit ang paborito mong exchange o wallet. Pagkatpos, gamitin mo ang Lightning wallet mo at i-click ang Manage button, at i-click ang "Refill".

Ano pang pwedeng gawin?

Pwede ka mag exchange ng ibang crypto gamit ang FixedFloat. Pwede ka rin bumili ng e-vouchers at iba pang bagay gamit ang BitRefill
